# UICollectionviewDemo 实现了大多数UICollectionview 在线上大多应用中的功能

## 规则网状列表

简单的网状列表展示，并且搭配简单的header和footer

## 不规则列表

不规则的列表展示，可以定制很多用tableview不容易实现的页面

## 自定义瀑布流

固定列数切宽度固定的瀑布流，仅提供实现思路

## 第三方瀑布流

引入第三方瀑布流的实现

## 一次多个的横向滑动

类似支付宝积福效果，一次可滑动多个元素

## 一次一个的横向滑动

类似nice首页的效果，一次可滑动一个元素

## 一次一个的（横向＋旋转）滑动

仅为了效果的展示
